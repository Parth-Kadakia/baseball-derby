Workflow

1. Pick a sheet ↑
2. + New Animation, name it (e.g. idle)
3. Drag on the sheet to box each frame — boxes get added to the active animation in order
4. Click a frame in the right panel to select it. Del removes it. Drag inside a selected box to move; drag a corner to resize.
5. Auto-slice row… over an empty area: drag a tall row, click the button, it auto-detects frames in that row
6. Export JSON — copies a `STRIPS_PRECUT` block to your clipboard for the game

Active sheet

Used as the top-level key in the exported JSON.
